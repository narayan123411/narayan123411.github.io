Power BI Data Visualization

In this project, I preprocessed the Covid-19 data, including datatype changes, and used the group by function to create targeted visualizations like county-specific and yearly graphs. I incorporated legends and other techniques to enhance the detail of these graphs. You can hover over the graphs for more insights into the data.

EMO VIBE

This project leverages an advanced Convolutional Neural Network, specifically the ResNet model, to analyze facial expressions and predict emotions with 95% accuracy. Based on the detected emotion, the system suggests music tracks that match the user's mood, updating recommendations each time while also considering the user's previous choices.

Flowchart Recognition

This project implements a pipeline for recognizing handwritten and computer-generated flowcharts using image preprocessing and shape-connector detection. It employs the Faster R-CNN model with a VGG-16 backbone, fine-tuned through hyperparameter adjustments. The dataset, originally focused on handwritten samples, was expanded with a computer-based dataset, enhancing the model's performance.

Automated Image Retrieval from Google Patents

This project involved downloading over 16,000 Google images from Google Patents, organizing them into folders named after each patent. I utilized Selenium in Python for automation, and to enhance efficiency, I implemented non-GUI multi-threading to significantly speed up the process.

Animated Map of COVID-19 Spread

This project presents an animated choropleth map illustrating the global spread of COVID-19 (Confirmed Cases). By processing monthly data, the animation tracks and visualizes the progression of cases across different countries, highlighting trends and patterns in the pandemic's evolution.

Spanish Sentiment Analysis with BERT

This project employs a BERT-based model to perform sentiment analysis on Spanish conversations. The model extracts key expressions and provides a score from 0 to 100, reflecting the overall sentiment and well-being of the individual based on their input.
